# TypTop-CPP #
An effort to port [typtop](https://github.com/rchatterjee/pam-typopw) to C++.

## Dependencies
* `cmake >= 3.6`
* Google `protobuf`
* Includes `cryptopp` (inside)
* pam-dev (in future)
